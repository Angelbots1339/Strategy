# Strategy
This is a repository for resources and tools related to robot game strategy and scouting.

## Resources
* [QRScout_config.json](QR%20Scout%20Config%20files/2025/QRScout_config.json): Config file for the [FRC2713](https://github.com/FRC2713/QRScout) QRScout [webapp](https://frc2713.github.io/QRScout/), used to initialize, hide, or otherwise configure fields.
* [FetchFirstAPI.gs](Google%20Apps%20Scripts/FetchFirstAPI.gs): Script for pulling data from the FRC Events API into Google Sheets.
